My second try : https://lapse.hackclub.com/timelapse/5chEBrlj1TVg

This project is just a circuit with 3 leds. The leds light up with a delay of approximatively 1second between each led and they fade when the switch is used. It works with 5v. The effect is nice and can be extended to the number of leds you want (you will just have to increase voltage and change the value of resistor and capacitor). For example, it can be used in a program where an infrared module detect someone and then the RC circuit begin to show the way in an stairway and then when the person leaved, the lights fade together.
